## iOS - How to Find System Setting

### Step-by-Step on iPhone

```
┌─────────────────────┐
│  1. Open Settings   │
│                     │
│  [Settings Icon]    │
│  Settings           │
└─────────────────────┘
         ↓
┌─────────────────────┐
│  2. Tap             │
│     Accessibility   │
│                     │
│  [♿ Icon]          │
│  Accessibility      │
└─────────────────────┘
         ↓
┌─────────────────────┐
│  3. Tap Motion      │
│                     │
│  [Motion Icon]      │
│  Motion             │
└─────────────────────┘
         ↓
┌─────────────────────┐
│  4. Toggle          │
│     Reduce Motion   │
│                     │
│  Reduce Motion      │
│  [────●] OFF        │
│                     │
│  or                 │
│                     │
│  Reduce Motion      │
│  [●────] ON         │
└─────────────────────┘
```

**Alternative (iOS 15+):**
- Pull down from home screen (Search)
- Type "Reduce Motion"
- Tap the result
- Toggle the setting

## Responsive Breakpoints

### Portrait Mode (typical: 375px width)
- Full-width layout
- Info boxes stack
- Toggle remains same size
- Text wraps naturally
- All elements visible without scrolling

### Landscape Mode (typical: 667px width)
- Slightly wider layout
- Info boxes still full-width
- Better use of horizontal space
- Toggle remains same size

### Small Screens (320px width)
- Tested on iPhone SE
- All elements remain usable
- Text remains readable
- Toggle remains 44px tall
- No horizontal scrolling
