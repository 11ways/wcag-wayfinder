# WCAG 2.2 Only - Important Notice

We are now only displaying the criteria **added in WCAG 2.2**.

However, to comply with **WCAG 2.2**, you must also include all the **WCAG 2.1 criteria**.

WCAG 2.2 does not change anything in WCAG 2.1; it simply adds a few new success criteria on top of the existing ones.

**To see all criteria required for WCAG 2.2 compliance**, please also check the "WCAG 2.1" version filter.
